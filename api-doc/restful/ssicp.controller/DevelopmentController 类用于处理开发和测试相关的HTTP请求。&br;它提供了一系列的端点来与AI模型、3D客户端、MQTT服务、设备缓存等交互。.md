# 调用AI模型。<br/>接收JSON格式的请求体，通过AI模型服务处理并返回处理结果。

## 请求信息

### 请求地址
```
http://192.168.1.38:8080/develop/aiModel?params=请求体中的参数，格式为JSON字符串。92
```

### 请求方法
```
POST
```

### 请求体类型
```
applicatin/json
```

## 入参
### 入参示例 (RequestBody)
```json
{
  "params": "请求体中的参数，格式为JSON字符串。100"
}
```


### 入参字段说明

| **字段** | **类型** | **必填** | **含义** | **其他参考信息** |
| -------- | -------- | -------- | -------- | -------- |
| params     | **String**     | 否  |  请求体中的参数，格式为JSON字符串。 |   |

## 出参
### 出参示例
```json
{
  "headers": {},
  "body": "溢漂洼秃鞘燎愚睛",
  "status": {}
}
```


### 出参字段说明

| **字段** | **类型**  | **含义** | **其他参考信息** |
| -------- | -------- | -------- | -------- |
| headers     | **HttpHeaders**    |   |   |
| body     | **String**    |   |   |
| status     | **Object**    |   |   |
